# Deep Multi-view Graph Clustering via Attribute-aware Bidirectional Structural Refinement and Pseudo-label Guided Multi-level Fusion
An official source code for paper "Deep Multi-view Graph Clustering via Attribute-aware Bidirectional Structural Refinement and Pseudo-label Guided Multi-level Fusion", accepted by ICML 2026. Any communications or issues are welcomed. Please contact zhaotianxiang0474@163.com.<br>
The authors of the paper: `Youqing Wang`, `Tianxiang Zhao`, `Mengyuan Xin`, `Ye Su`, `Jiapu Wang`, `Tengfei Liu`, `Junbin Gao`, `Jipeng Guo` (Corresponding author).<br>
### Overview
We propose a novel Deep Multi-view Graph Clustering (**APGC**), incorporating **A**ttribute-aware **B**idirectional **S**tructural **R**efinement (**ABSR**) and **P**seudo-label **G**uided **M**ulti-level **F**usion (**PGMF**). The overall framework of **APGC** is illustrated in Fig. 1.<br>
